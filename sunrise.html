Sunrise generator

Everyone loves a sunrise, but everyone also loves a decent lie-in. This paradox needed a solution, and so I decided to make a Sunrise Generator, so you can enjoy a beautiful sunrise any time of day, and save it to use as wallpaper.

This self-initiated project was also the perfect chance for me to learn more about image pixel-array manipulation and working with colour.

Make yourself a sunrise.

Colour Interpolation

I wanted to create a abstract gradient result which had the random striped progression you see in the sky when the sun rises.

To create this effect I first wrote a method which would linearly interpolate between two colours, and then for each row, instead of adding the same control value each time (to make a smooth gradient), I use a new random number for each row, which is usually a plus jump towards the end colour, but can also sometimes move backwards towards the original colour with a minus. Some rows also have a larger colour jump than others.

Noise

To create a less ‘digital’ and softer feel to the image, I then added some noise to the image by looping through the pixel array of the canvas and adding or subtracting from the R, G and B values. I had noticed when looking at reference photos that noise tended to be higher in lighter areas, and so I added some logic which took this into account and turned down noise on darker pixels.

Randomisation

On each new sunrise, all the values and colours (colour-picked from sunrise photos) are randomised so the effect is different each time. I spent a lot of time experimenting with different colours and min/max values for the random numbers until I hit the sweet-spot.

First Version in Processing

I made my first version of the Sunrise Generator in late 2013 as an exercise to learn about working with and manipulating pixel-arrays in Processing.

I experimented a lot with different pattern-making techniques before the Sunrise Generator, and although I got results I was happy with, it was very slow and limited.

Take a look at the Processing version.

I got rid of the blurred border. It was interesting to make, but unnecesarry.

Rebuilding in JavaScript on Canvas

In summer 2014 I decided to revisit the Sunrise Generator and rebuild it from scratch in JavaScript on Canvas.

I simplified and refined some of the techniques I had developed, and built upon the idea to create a website containing the Sunrise Generator. Every second a new sunrise is made, and visitors can save the sunrise as an image to use as a wallpaper.

I wanted to generate new sunrises quickly, to encourage the impulse of clicking-to-save when you see one which instantly speaks to you. But when it's gone, it's gone.
